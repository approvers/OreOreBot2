# OreOreBot2 Documents へようこそ

## はらちょとは

> はらちょは 2020 年 2 月ころ (サーバーが発足してすぐ) に限界開発鯖で開発が始まった Bot です. アプリケーションとしての正式名称は OreOreBot です.
> ----- [MikuroXina - はらちょ(ネタ Bot)に詰め込まれたすごい技術](https://qiita.com/MikuroXina/items/6b58f15d5f12782f7274#%E3%81%AF%E3%82%89%E3%81%A1%E3%82%87%E3%81%A8%E3%81%AF)

当初 discord.py + Python で開発されたのが [はらちょ](https://github.com/approvers/OreOreBot)(OreOreBot) です。

Discord で動作する Bot で、くだらない機能が盛りだくさん。つまりネタ Bot です。

## 新生はらちょとは

そんなはらちょですが、2021 年に入ってから開発が停滞し、あまりのコードの見通しの悪さに最初 Go 言語のでの再実装も予定されましたが失脚。

2021 年 12 月に [discord.py の開発停止](https://gist.github.com/Rapptz/4a2f62751b9600a31a0d3c78100287f1)(今は開発が再開しています。) が発表されたことで、はらちょが永眠してしまうのではないかという懸念が出てきました。

ということで 2022 年 1 月からはらちょの開発が再開され、discord.py から discord.js に移行することになりました。通称**はらちょぶっ殺しプロジェクト**です。

そんなこんなで [はらちょ](#はらちょとは) を discord.js + TypeScript で再実装したのが [新生はらちょ](https://github.com/approvers/OreOreBot2) になります。
