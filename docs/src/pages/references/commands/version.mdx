# バージョン表示

<CommandArgs versionAvailableFrom="v1.16.0" commandName="version" />

現在のはらちょのバージョンを表示します。

はらちょのバージョンはバグレポートの提出に必要です。

<Callout type="info" emoji="💡">バージョンは `!version` コマンド以外に **起動ログ、アクティビティ欄** にも表示されます。</Callout>
