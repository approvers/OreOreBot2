ここだけの話: コウメ太夫構文

コウメ太夫の元々のネタを完全に再現するため、スペース等を本物と合わせています。

[eslint の警告まで BAN ](https://github.com/approvers/OreOreBot2/blob/main/src/service/command/meme/koume.ts#L10-L15)されるほどの厳格さです。[あの RC](https://github.com/approvers/OreOreBot2/pull/584#discussion_r1032444190) は今でも意味がわかりません。

```ts
generate(args) {
  const [option1, option2] = args.body.split(' ');
  // Reason: 構文とコウメ太夫に敬意を払い、元ネタを尊重することから全角スペースを使用したいのでeslintの警告をBANします。
  // eslint-disable-next-line no-irregular-whitespace
  return `${option1}と思ったら〜♪\n\n${option2}でした〜♪\n\nチクショー！！　#まいにちチクショー`;
}
```
