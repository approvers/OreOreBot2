# Party 一葉

<CommandArgs versionAvailableFrom="v1.0.0" commandName="kaere" />

---

あなたのボイスチャンネルをパーティー会場に！引数なしで即時発動

<Callout type="warning">

この機能には FFmpeg が必要です。

</Callout>

<Callout type="info">

実行者がボイスチャンネルにいないとこのコマンドは安全装置によって封じ込められます。

予約機能についても同様で、実行時間にボイスチャンネルにいないと実行されません。

</Callout>

---

### `!party status`

[ゲリラモード](#ゲリラモード)の設定状況を表示します。

### `!party set <id>`

指定した ID の BGM に設定します。

利用可能な BGM は[はらちょが提供する BGM](#はらちょが提供する-bgm)を確認してください。

### `!party enable`

[ゲリラモード](#ゲリラモード)を有効にします。

### `!party disable`

[ゲリラモード](#ゲリラモード)を無効にします。

### `!party time [時間]`

[ゲリラモード](#ゲリラモード)の実行時間を設定します。

実行時間は分単位で指定します。毎時間その時間にゲリラを実行します。

例: `!party time 30` で 30 分毎にゲリラを実行します。

### ゲリラモード

ゲリラモードは、ボイスチャンネルにいる人間に**不意打ちで**押しかけてパーティーを開く機能です。

ゲリラを行う時間は `!party time` で設定することが出来ます。設定していない場合はランダムな時間に実行されます。
