# VC 接続 Diff

<FeatureBadge>ログ</FeatureBadge>,<VersionBadge>v1.0.0</VersionBadge>

---

VC に接続したメンバーがいたら `#無法地帯` にログを送信します。

<Callout type="warning" emoji="⚠️">
  はらちょに **チャンネル閲覧権限** が付与されていないチャンネルに関するログは送信されません。
  ログを受け取りたいチャンネルには権限を与えるようにしてください。
</Callout>
