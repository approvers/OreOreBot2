---
title: 'パッシブ機能'
---

このボットのパッシブ機能は、以下の種類に分けられます。

- メッセージへの操作を監視するもの
  - [メッセージの編集を検知して報告する](/references/features/edit-diff)
  - [メッセージの削除を検知して報告する](/references/features/delete-diff)
- サーバー内での行動のログ作成
  - [ボイスチャンネルの入退室を検知して報告する](/references/features/vc-diff)
  - [カスタム絵文字の作成を検知して報告する](/references/features/emoji-create-log)
  - [スタンプの作成を検知して報告する](/references/features/sticker-create-log)
- メッセージ本文の収集
  - [Bold-italic なメッセージに過剰反応する](/references/features/bold-italic)
  - [特定パターンのメッセージを収集/閲覧する](/references/features/typo)
- その他
  - [設定時刻にメッセージを送信する](/references/features/time-signal)
  - [かわえもんにすべてのロールを付与する](/references/features/kawaemon)

ここにドキュメント化されていない機能はメニューにあるそれぞれの子のページから閲覧できます。
