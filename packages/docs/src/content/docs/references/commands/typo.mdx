import { CommandArgs } from '@/organisms/command-args';

<CommandArgs
  availableFrom="v1.0.0"
  names={['typo']}
  args={[
    {
      name: 'by <ユーザーID>',
      about: '指定したユーザーの Typo を表示します。',
      defaultValue: '指定なし',
      availableFrom: 'v1.14.0'
    }
  ]}
/>

その日に記録された、あなたの Typo (`だカス` で終わるメッセージ) を表示します。

詳しくは [今日の Typo 機能のドキュメント](/references/features/typo) をご覧ください。

## 他のユーザーの Typo を見る

`!typo by <ユーザーID>` と実行すると、指定ユーザーの今日の Typo を確認できます。
