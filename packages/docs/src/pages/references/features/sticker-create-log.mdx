# スタンプ作成ログ

<FeatureBadge>ログ</FeatureBadge>,<VersionBadge>v1.51.0</VersionBadge>

---

スタンプを作成するとログを送信します。

作成されたスタンプについて、以下の情報を表示します。

### 表示される情報

- 作成者
- 名前
- 画像
- 説明
- 関連した絵文字
    - Autocomplete用。絵文字を送信すると候補として表示されます。
