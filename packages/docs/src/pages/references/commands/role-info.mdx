# ロール情報表示

<CommandArgs
  versionAvailableFrom="v1.21.0"
  commandName="roleinfo/role"
  args={[
    {
      name: 'ロールID',
      about:
        '情報を表示したいロールのIDです。ロールのメンションを指定しないようにしてください。'
    }
  ]}
/>

指定したロールについて、以下の情報を表示します。

## 表示される情報

| 項目         | 説明                                                                        |
| ------------ | --------------------------------------------------------------------------- |
| ID           | ロールの ID                                                                 |
| 所属人数     | そのロールを持つメンバーの数                                                |
| ポジション   | ロールリスト内での位置 <br/> (Discord API 上では、最終尾から数えるため注意) |
| カラーコード | そのロールのカラーコード                                                    |
| 作成日時     | そのロールが作成された日時と経過時間                                        |

## ロールの絵文字

指定したロールに対して Role Emoji が設定されていない場合はその絵文字が表示されます。

Unicode、カスタム絵文字かで挙動が大きく変わります。

| 絵文字         | 挙動                                      |
| -------------- | ----------------------------------------- |
| Unicode        | `アイコン` というフィールドの値として表示 |
| カスタム絵文字 | 絵文字の画像そのものを表示                |
