# はじめに

新生はらちょ (OreOreBot2) は限界開発鯖で利用できる Discord Bot です。このボットには主に以下の 2 種類の機能があります。

- <Link to="./commands/">ユーザーの命令で動作するコマンド機能</Link>
- <Link to="./features/">サーバー内での行動に反応するパッシブ機能</Link>

詳しくはそれぞれのページをご覧ください。
