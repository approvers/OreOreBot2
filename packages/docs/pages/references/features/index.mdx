# パッシブ機能

このボットのパッシブ機能は、以下の種類に分けられます。

- メッセージへの操作を監視するもの
  - <Link to="./edit-diff/">メッセージの編集を検知して報告する</Link>
  - <Link to="./delete-diff/">メッセージの削除を検知して報告する</Link>
- サーバー内での行動のログ作成
  - <Link to="./vc-diff/">ボイスチャンネルの入退室を検知して報告する</Link>
  - <Link to="./emoji-create-log/">カスタム絵文字の作成を検知して報告する</Link>
  - <Link to="./sticker-create-log/">スタンプの作成を検知して報告する</Link>
- メッセージ本文の収集
  - <Link to="./bold-italic/">Bold-italic なメッセージに過剰反応する</Link>
  - <Link to="./typo/">特定パターンのメッセージを収集/閲覧する</Link>
- その他
  - <Link to="./time-signal/">設定時刻にメッセージを送信する</Link>
  - <Link to="./kawaemon/">かわえもんにすべてのロールを付与する</Link>

ここにドキュメント化されていない機能はメニューにあるそれぞれの子のページから閲覧できます。
